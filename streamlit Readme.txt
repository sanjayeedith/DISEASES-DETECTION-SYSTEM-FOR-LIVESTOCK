# Livestock Health Prediction Project

## Overview

This project aims to predict the health status of livestock animals based on a set of input features. It includes a Streamlit web application for user interaction and a trained Decision Tree model for making predictions.

## Project Structure

- `app.py`: The Streamlit web application.
- `animal_decision_tree_model.joblib`: The trained Decision Tree model file.
- `generate_dataset.py`: Script to generate a synthetic dataset for training.
- `requirements.txt`: List of Python dependencies.

## Setup

1. Install the required dependencies:

 -- pip install streamlit
 cmd:
  	streamlit run app.py
